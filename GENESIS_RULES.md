# GENESIS RULES

No system moves to implementation until these exist:

1. SYSTEM_CONSTITUTION.md
2. THREAT_MODEL.md
3. CONTROL_AND_PERMISSIONS_MAP.md
4. ASSET_AND_VALUE_FLOW.md
5. PROOF_REQUIREMENTS.md
6. LAUNCH_BLOCKERS.md

No system is called production-ready until these exist:

1. VERIFICATION_MATRIX.md
2. CLAIMS_TO_PROOF_MAP.md
3. PUBLIC_TRANSPARENCY_PAGE.md
4. PUBLIC_RISK_DISCLOSURE.md
5. OPERATOR_RUNBOOK.md
6. MASTER_SYSTEM_REPORT.md

Blocked launch conditions:
- unclear ownership
- unclear admin roles
- unclear treasury control
- unverifiable public claims
- missing rollback/recovery path
- missing observability
- missing operator documentation
